About Me

학원에서 게임 프로그래밍으로 코딩을 배움으로써 코딩으로 구현하는 재미를 알았습니다. 처음 게임을 만들기 시작할 때 구현 불가능해 보이는 것들이 하나씩 완성되면서 쉽게 포기하지 않으면 만들 수 있구나, 내 머릿속에 있는 걸 코드로 짜내서 구현할 수 있다는 걸 느꼈고 코드가 입력될 때마다 앞에 보이는 것들이 살아있는 느낌이 드는 것 또한 좋았습니다. 무엇보다 완성할 때의 그 성취감은 엄청나게 중독적이었습니다.

제가 게임 프로그래밍으로 만든 프로젝트는 타이탄소울즈라는 모작 게임 그리고 자작 게임을 만들어 냈습니다. 모작 게임은 C++과 DirectX를 통해 프레임워크가 없는 하드한 코딩이었고 생소한 개념인 벡터를 새로 배웠습니다. 자작게임은 엔진도 새로 익혀야 하고 코딩보다는 디자인과 구성을 혼자서 생각해야 하는 어려움이 있었습니다. 두 프로젝트를 진행하면서 힘들었지만, 누구의 도움도 없이 만들었기에 자신감과 차분하게 시도한다면 구현할 수 있다는 인내심을 배웠고, 개발자는 협업뿐만 아니라 혼자서 고생하며 자기만의 지식으로 가공하고 기술을 쌓는 것이 중요하다고 생각했습니다

앞의 프로젝트를 바탕으로 웹 프로그래밍을 시작했습니다. c++과 java는 객체지향 언어인것이 비슷했지만 새로 공부할 게 무척이나 많았습니다. 스프링과 HTML Javascript를 익혀 간단한 게시판을 만들어보면서 MVC 체계를 익히는 데 도움이 많이 되었고, 바로 간단한 운동 웹 애플리케이션을 만들기 시작했습니다. 게시판과 다르게 생소한 개념들이 많이 있었고 특히 JSON 데이터들을 컨트롤러마다 이동시켜 뷰에 보이게 해주는 것이 엄청나게 까다로웠습니다. 하지만 전의 경험을 통해 차분히 짤 수 있게 된 로직과 JSON 데이터들을 Javascript와 jQuery의 셀럭터를 통해 구성요소들을 동적으로 생성해냈고 결과적으로 생각한 것들을 구현할 수 있게 되었습니다. 또한, AJAX 와 DB, JSON 데이터 개념들을 써 볼 수 있는 경험이 생겨 다양한 기술을 익히게 되었습니다. 혼자 만들 때의 막막함과 어려움은 항상 있었지만, 책상에 앉아서 알고리즘을 생각해내는 것이 재미있었고 무엇을 처음 배우고 시작할 때 앞선 결과물들이 있기에 집중한다면 무조건 해결할 수 있고 그만큼 실력이 늘겠다는 생각이 있기에 항상 자신이 있습니다.

그렇다고 항상 책상에만 앉아서 끙끙거리며 힘들어하는 게 아닙니다. 자기 관리를 좋아하기 때문에 운동을 좋아하고 10년 가까이 꾸준히 해왔습니다. 그래서 머릿속이 복잡해질 때면 운동으로 에너지를 써왔습니다. 이렇듯 저는 항상 긍정적이고 활력이 넘칩니다. 그렇기에 새로운 것들을 접할 때 그다지 어렵지 않았습니다. 게임 프로그래밍에서 웹 프로그래밍으로 넘어갈 때 설레임이 앞섰습니다. 새로운 것은 저에게 도전이자 다른 방향으로 나아가는 또 다른 인생의 이정표라고 생각하기 때문에 항상 재밌습니다. 이와 똑같이 새로운 장소, 새로운 사람을 만날 때도 항상 흥미롭고 약간의 긴장감과 어색함이 탐험하는 느낌을 줍니다. 저는 모험을 좋아합니다. 새로운 장소, 그곳에서의 만남과 배움, 그리고 이 모든 게 저를 어제와 또 다른 저로 만들어줄 것으로 생각하기에 도전하고 싶습니다.
